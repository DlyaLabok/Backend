🌐 HTTP Server Basics

Демонстрація Node.js http модуля

📚 Що демонструється:

  • http.createServer() — створення сервера
  • Request — об'єкт запиту (method, url, headers)
  • Response — об'єкт відповіді (writeHead, end)
  • URL Routing — маршрутизація запитів
  • Query String — параметри запиту
  • POST Data — обробка POST запитів
  • Static Files — роздача статики
  • JSON API — REST API endpoints

📖 Про сайт ⏰ Поточний час 👋 Привітання 👤 Користувач #42 📝 Форма 📦 JSON API

📋 API Endpoints:

# Отримати час
curl http://localhost:3000/time

# Привітання
curl "http://localhost:3000/hello?name=Марія"

# Список користувачів
curl http://localhost:3000/api/users

# POST запит
curl -X POST -H "Content-Type: application/json" \
     -d '{"name":"Test"}' \
     http://localhost:3000/api/data
